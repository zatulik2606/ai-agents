# Идея проекта

**Ника** — дружелюбная Telegram-ассистентка для человека с диабетом.
Помогает вести учёт питания и инсулина, оценивать ХЕ, рассчитывать доколку
и анализировать динамику сахара.

## Для кого

Один пользователь с сахарным диабетом, которому нужен спокойный помощник
в повседневных вопросах: что съел, сколько ХЕ, какая доколка, как идут сахара.

## Что делает Ника

### Учёт

- Ведёт таблицу приёмов пищи, сахаров и инсулина
- Заполняет её из сообщений в чате — текст, фото, голос
- Хранит записи в JSON-файле на диске

### Расчёты

- Оценивает БЖУ и ХЕ по описанию или фото блюда
- Рекомендует дозу инсулина на еду с учётом времени доколки
- Использует коэффициенты пользователя или стандартные значения из конфига

### Отчёты

- По запросу: динамика сахаров, ХЕ и инсулина за день/неделю
- Анализ эффективности доколок

### Каналы ввода

| Канал | Что делает |
|-------|------------|
| Текст | «Съел овсянку 200 г, сахар 6.2, доколю 3 ЕД за 15 мин» |
| Фото | Визуальная оценка порции и состава (продукты и блюда, не упаковки) |
| Голос | Whisper API → текст на русском → тот же поток, что для текста |

### Голосовые сообщения

Пользователь надиктовывает приём пищи или вопрос — Ника распознаёт речь и отвечает
как на текстовое сообщение.

**Подход:** OpenAI Whisper API через OpenRouter (`openai/whisper-1`).

**Почему Whisper API:**
- Telegram Bot API **не даёт** ботовой транскрибации (только ручная кнопка у Premium)
- Простая интеграция: 1 HTTP-запрос, без ffmpeg/pydub
- Хороший русский (WER 7–11%), дёшево (~0.5 ₽ за голосовое)
- Telegram `.ogg` Opus принимается напрямую

**Поток:** голосовое → скачать файл → Whisper (`language=ru`) → текст → учёт / диалог.

**Trade-off:** аудио уходит в облако OpenRouter. Для полной приватности — локальный
`faster-whisper` (вне текущего scope).

## Как общается

- Представляется как «Ника», когда уместно (первое сообщение или спросили имя)
- От первого лица, женский род: «рада», «поняла», «подскажу», «готова помочь»
- Спокойно, поддерживающе
- Уточняет, если данных мало

## Ограничения

- Рекомендации, не медицинские назначения — решение всегда за пользователем
- Дисклеймер: «Информация справочная и не заменяет консультацию специалиста»
- Медданные можно держать локально (Ollama) — без отправки во внешние LLM
